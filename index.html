    Feb 27, 2019#Network

    网络性能指标

    • 带宽

      表示链路的最大传输速率，单位是 b/s （ 比特 / 秒 ) 。常见的带宽有 1000M、10G、40G、100G 等。

    • 吞吐量

      表示没有丢包时的最大数据传输速率，单位通常是 b/s （ 比特 / 秒 ) 或者 B/s ( 字节 / 秒 ) 。

    • 延时

      表示从网络请求发出后，一直到收到远端响应，所需要的时间延迟。

    • PPS ( Packet Per Second )
      表示以网络包为单位的传输速率。PPS 通常用来评估网络的转发能力。
