Leetcode 283 Move Zeroes

Feb 23, 2019#Alogorithms

题目来源：https://leetcode.com/problems/move-zeroes/

标记难度：Easy

提交次数：1/N

代码效率：100%

题意

给定一个整形数组，将0都移动到最后，保持其他数组相对位置不变

分析

一个快指针，一个慢指针，快指针指向不为0的元素，快慢指针值交换，同时向后移动一位。这样可以保证下面两个条件：

  • 快慢指针之间的元素都是0
  • 慢指针之前的元素都不为0
